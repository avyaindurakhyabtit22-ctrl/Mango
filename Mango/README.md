# Mango 🍋
AI-powered manga translation web app built with Next.js, Tailwind, and Genkit.

## 🚀 Features
- Image Upload & Batch Processing (up to 30 images).
- AI-Powered OCR for manga speech bubbles.
- Real-time Translation into chosen language.
- Dynamic Text Overlay with coordinates.
- Visual Preview (original vs translated).
- Progress Tracking with a progress bar.
- Error Handling with graceful fallbacks.

## 🛠 Tech Stack
- Next.js
- Tailwind CSS
- Genkit
- Google Gemini
- shadcn/ui

## 📂 AI Flows
- Extract Text → OCR from manga images.
- Translate Text → Translates extracted text.
- Improve Translation → Refines translation quality.
- Suggest Language → Suggests best target language.
- Text-to-Speech → Converts text to audio.
- Multi-Speaker TTS → Generates multi-speaker audio.
